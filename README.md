# 🎬 AI Movie Recommendation System

An AI-powered movie recommender that suggests movies based on your mood, genre, and ratings.

---

## 🚀 Features
- 🎭 Genre-based filtering  
- 😊 Sentiment Analysis (TextBlob)  
- ⭐ Rating-based recommendations  
- 🧠 Keyword matching (AI logic)  
- 🎲 Surprise movie option  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- TextBlob  
- Colorama
